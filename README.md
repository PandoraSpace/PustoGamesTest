# PustoGamesTest
Тестовое сделал Магдеев Тимур.
Использовал DOTween для анимации движения стрелок часов. 
В проекте есть Bootstrap - точка входа в приложение. 
UI написан на MVVM подходе.
Для получения времени с сервера использовал GET запрос UnityWebRequest и newtonsoft-json библиотеку для десериализации ключ-значения.
----------
Из фишек реализовна кнопка Edit: мышкой можно подвигать стрелки для выставления времени.
Тестовое сделано за ~2 вечера.
